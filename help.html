FAQ

How to run a QuickScan?
Send a token contract address, DexScreener URL or a pair address to the bot. Example: 0x6982508145454Ce325dDbE47a25d4ec3d2311933.
What do the Δ buttons (5m / 1h / 6h / 24h) do?
They switch the time range for price deltas in the QuickScan reply.
How do I open Report (HTML) / Why / Why++?
Tap the corresponding inline buttons under the scan result. The report opens as a web page.
What is LP lock (lite)?
We show a compact LP lock block with links to UNCX / TeamFinance and an approximate unlock date if we can detect it.
My report shows outdated stats.
Reports cache for a short time to improve speed. Re-run the scan after a minute or use the Δ buttons to refresh.
