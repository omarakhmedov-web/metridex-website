Ончейн‑метрики риска для DeFi и Telegram

QuickScan выдаёт мгновенные сигналы риска, анализ ликвидности и держателей, флаги админ‑прав и проверки домена (WHOIS / SSL / Wayback). По запросу доступен развёрнутый HTML‑отчёт и шаринг.

Ключевые функции

  • Δ‑кнопки 5м / 1ч / 6ч / 24ч
  • Концентрация ликвидности и топ‑держателей
  • Флаги админ‑прав и прокси‑контрактов
  • Домен: WHOIS, SSL, Wayback‑снимки
  • Шаринг HTML‑отчёта, copy‑CA, открытие в Scan/DEX
  • Watchlist и Alerts (lite): оповещения по цене, ликвидности, изменениям держателей/LP-lock и важным флагам.

Зачем это нужно

  • Быстрый скоринг для трейдеров и админов каналов
  • Снижение риска скама за счёт автоматических проверок
  • White‑label для каналов и launchpad‑ов

Тарифы (пример)

  • Дневной — $9
  • Pro — $29/мес
  • Teams — $99/мес
  • Deep Report — от $3

Технологии и интеграции

  • Мультичейн (ETH / BSC / Polygon)
  • DexScreener, блок‑сканеры, WHOIS/SSL API
  • Render + Cloudflare Worker proxy
